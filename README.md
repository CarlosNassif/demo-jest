## Instalação

Necessário [Node.js](https://nodejs.org/) v16+ para rodar.

Instala as dependências e roda os testes

```sh
cd demo-jest
npm i
npm run test
```

Com cobertura de código gerando relatório

```sh
npm run test --collect-coverage
```
